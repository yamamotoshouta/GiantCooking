# Genies SDK - Avatar

## Overview

The Genies Avatar SDK enables Unity developers to integrate Genies avatars into their applications. It provides APIs for user authentication, avatar loading, and session management — allowing you to bring interoperable user-owned avatars into your experience with minimal setup.

## Getting Started

The fastest way to get up and running is through the **Genies SDK Bootstrap Wizard**:

**Tools > Genies > SDK Bootstrap Wizard**

The wizard automatically checks your project prerequisites, configures required settings, and walks you through installing and setting up the SDK.

## Documentation
