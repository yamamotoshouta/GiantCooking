# AntiGravity VR Project

物理演算をベースとしたVR剣戟押し出しアクションゲームのコアシステムです。

## フォルダ構成
- `Editor/`: 自動セットアップ用のエディタ拡張スクリプト
- `Scripts/`: ゲームロジック
    - `Core/`: ゲームマネージャー、剣の物理ロジック
    - `UI/`: VR内ゲージUI
    - `Effects/`: 剣の発光、酔い防止エフェクト
    - `System/`: 落下判定など
- `Prefabs/`: 将来的にプレハブを保存する場所
- `Materials/`: ゲーム内マテリアル

## 使い方
1. Unityエディタの上部メニューから **[AntiGravity] > [Create VR Game Scene]** を選択してください。
2. 必要なオブジェクト（プレイヤー、敵、剣、ステージ）が自動的に生成されます。
3. Playボタンを押して、VRヘッドセットで体験してください。

## 主要な仕組み
- **物理はじき**: 剣同士がぶつかると、Rigidbodyの力によって物理的に弾き合います。
- **極ゲージ**: 剣戟を繰り返すとゲージが溜まり、剣が発光します。
- **一閃**: ゲージMAX時に相手を吹き飛ばす強力な一撃を放てます。
- **フォールアウト**: ステージから落とされるとリスタートします。
